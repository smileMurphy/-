## SpaceDesk
### 描述：两个笔记本之间通过局域网进行屏幕共享、扩展
### 地址：https://www.spacedesk.net/
#### 提示：浏览器访问链接为http://spacedesk.ph/html5viewer/
</br> 

## TeamViewer
### 描述：用于两台电脑之间远程控制，具有内网穿透功能，常用来远程控制服务器
### 地址：https://www.teamviewer.com/
#### 提示：安装时选择非商业用途，即可免费使用
</br>
